## 使用边界

- 查当前已生效的行为契约：从本索引进入对应 `spec.md`。
- 查尚未归档、可能仍含未同步 delta 的提案：查看 [Active Changes](../changes/README.md)。
- 查历史决策与归档证据：查看 [Archived Changes](../changes/archive/README.md)。
- 本页只做导航，不复制 requirement 正文；capability 语义以各自 `spec.md` 为准。
